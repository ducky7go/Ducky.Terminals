# Terminal-Oberfläche - Universelle Konsole für Mods
Repository: https://github.com/ducky7go/Ducky.Terminals
## Was ist die Terminal-Oberfläche?
Die Terminal-Oberfläche ist ein universelles Konsolen-Framework für Escape from Duckkov. Sie bietet eine zentralisierte Befehlszeilenschnittstelle, in der alle kompatiblen Mods ihre Befehle registrieren können, wodurch ein einheitliches Steuerungszentrum für Mod-Verwaltung und -Operationen entsteht.
## Verwendung
### Terminal öffnen
1. Starten Sie das Spiel und betreten Sie eine beliebige Szene
2. Bewegen Sie Ihre Maus in die untere linke Ecke des Bildschirms
3. Ein Auslösebereich erscheint - klicken Sie darauf, um das Terminal zu öffnen
4. Das Terminal gleitet von links herein und nimmt 50% des Bildschirms ein
### Mod-Anbieter auswählen
1. Klicken Sie auf die Schaltfläche "Select Provider" unten links im Terminal
2. Eine Liste aller verbundenen Mods erscheint
3. Klicken Sie auf einen beliebigen Mod, um ihn als aktiven Anbieter auszuwählen
4. Alternativ können Sie # gefolgt von der kurzen Mod-ID eingeben (z.B. #816, #dem), um Anbieter schnell zu filtern
### Befehle ausführen
1. Wählen Sie einen Anbieter aus, geben Sie dann Befehle in das Eingabefeld unten ein
2. Drücken Sie die Eingabetaste, um den Befehl auszuführen
3. Die Ergebnisse werden im scrollbaren Nachrichtenbereich darüber angezeigt
4. Verwenden Sie help oder /?, um verfügbare Befehle für den ausgewählten Anbieter anzuzeigen
