# ターミナルUI - Mod用ユニバーサルコンソールインターフェース
リポジトリ：https://github.com/ducky7go/Ducky.Terminals
## ターミナルUIとは？
ターミナルUIは、Escape from Duckkov用のユニバーサルコンソールフレームワークです。互換性のあるすべてのModがコマンドを登録できる一元化されたコマンドラインインターフェースを提供し、Mod管理と操作のための統合されたコントロールセンターを作成します。
## 使用方法
### ターミナルを開く
1. ゲームを起動して任意のシーンに入ります
2. マウスを画面の左下隅に移動します
3. トリガーエリアが表示されます - クリックしてターミナルを開きます
4. ターミナルが左側からスライドし、画面の50%を占有します
### Modプロバイダーを選択する
1. ターミナルの左下にある「Select Provider」ボタンをクリックします
2. 接続されているすべてのModのリストが表示されます
3. 任意のModをクリックしてアクティブプロバイダーとして選択します
4. あるいは、Modの短いID（例：#816、#dem）の前に#を入力してプロバイダーを素早くフィルタリングできます
### コマンドを実行する
1. プロバイダーを選択後、下部の入力フィールドにコマンドを入力します
2. Enterキーを押してコマンドを実行します
3. 結果が上部のスクロール可能なメッセージエリアに表示されます
4. helpまたは/?を使用して、選択されたプロバイダーの利用可能なコマンドを表示します

あなたのModをターミナルUIと統合したいですか？完全な統合ガイド、APIドキュメント、コード例についてはhttps://github.com/ducky7go/Ducky.Terminalsを訪問してください。SDKベース（推奨）と手動統合の両方の方法をサポートしています。
