# 터미널 UI - 모드용 범용 콘솔 인터페이스
저장소: https://github.com/ducky7go/Ducky.Terminals
## 터미널 UI란?
터미널 UI는 Escape from Duckkov용 범용 콘솔 프레임워크입니다. 호환되는 모든 모드가 명령어를 등록할 수 있는 중앙 집중식 명령줄 인터페이스를 제공하여 모드 관리 및 작업을 위한 통합 제어 센터를 만듭니다.
## 사용 방법
### 터미널 열기
1. 게임을 시작하고 임의의 장면에 들어갑니다
2. 마우스를 화면 왼쪽 하단으로 이동합니다
3. 트리거 영역이 나타납니다 - 클릭하여 터미널을 엽니다
4. 터미널이 왼쪽에서 미끄러져 나와 화면의 50%를 차지합니다
### 모드 공급자 선택
1. 터미널 왼쪽 하단의 "Select Provider" 버튼을 클릭합니다
2. 연결된 모든 모드의 목록이 나타납니다
3. 임의의 모드를 클릭하여 활성 공급자로 선택합니다
4. 또는 모드의 짧은 ID(예: #816, #dem) 앞에 #을 입력하여 공급자를 빠르게 필터링할 수 있습니다
### 명령어 실행
1. 공급자를 선택한 후 하단 입력 필드에 명령어를 입력합니다
2. Enter 키를 눌러 명령어를 실행합니다
3. 결과가 위의 스크롤 가능한 메시지 영역에 표시됩니다
4. help 또는 /?를 사용하여 선택된 공급자의 사용 가능한 명령어를 봅니다
