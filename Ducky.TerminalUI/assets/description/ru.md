# Терминальный Интерфейс - Универсальная Консоль для Модов
Репозиторий: https://github.com/ducky7go/Ducky.Terminals
## Что такое Терминальный Интерфейс?
Терминальный Интерфейс - это универсальный фреймворк консоли для Escape from Duckkov. Он предоставляет централизованный интерфейс командной строки, где все совместимые моды могут регистрировать свои команды, создавая единый центр управления для модификаций и операций.
## Как Использовать
### Открытие Терминала
1. Запустите игру и войдите в любую сцену
2. Переместите мышь в нижний левый угол экрана
3. Появится область активации - нажмите на нее, чтобы открыть терминал
4. Терминал выдвигается слева, занимая 50% экрана
### Выбор Поставщика Мода
1. Нажмите кнопку "Select Provider" в нижнем левом углу терминала
2. Появится список всех подключенных модов
3. Нажмите на любой мод, чтобы выбрать его как активного поставщика
4. Альтернативно, введите # с коротким ID мода (например, #816, #dem) для быстрой фильтрации поставщиков
### Выполнение Команд
1. После выбора поставщика введите команды в поле ввода внизу
2. Нажмите Enter для выполнения команды
3. Результаты отобразятся в прокручиваемой области сообщений выше
4. Используйте help или /? для просмотра доступных команд для выбранного поставщика
