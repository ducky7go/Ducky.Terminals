# Interfaz de Terminal - Consola Universal para Mods
Repositorio: https://github.com/ducky7go/Ducky.Terminals
## ¿Qué es la Interfaz de Terminal?
La Interfaz de Terminal es un framework de consola universal para Escape from Duckkov. Proporciona una interfaz de línea de comandos centralizada donde todos los mods compatibles pueden registrar sus comandos, creando un centro de control unificado para la gestión y operaciones de mods.
## Cómo Usar
### Abrir el Terminal
1. Inicia el juego y entra en cualquier escena
2. Mueve tu ratón a la esquina inferior izquierda de la pantalla
3. Aparecerá un área de activación - haz clic para abrir el terminal
4. El terminal se desliza desde la izquierda, ocupando el 50% de la pantalla
### Seleccionar un Proveedor de Mod
1. Haz clic en el botón "Select Provider" en la esquina inferior izquierda del terminal
2. Aparecerá una lista de todos los mods conectados
3. Haz clic en cualquier mod para seleccionarlo como proveedor activo
4. Alternativamente, escribe # seguido del ID corto del mod (ej. #816, #dem) para filtrar proveedores rápidamente
### Ejecutar Comandos
1. Después de seleccionar un proveedor, escribe comandos en el campo de entrada inferior
2. Presiona Enter para ejecutar el comando
3. Los resultados se mostrarán en el área de mensajes desplazable arriba
4. Usa help o /? para ver los comandos disponibles para el proveedor seleccionado
