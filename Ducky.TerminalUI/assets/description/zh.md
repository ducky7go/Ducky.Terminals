# 终端界面 - 模组通用控制台
仓库地址：https://github.com/ducky7go/Ducky.Terminals
## 什么是终端界面？
终端界面是为《逃离鸭科夫》打造的通用控制台框架。它提供了一个集中的命令行界面，所有兼容模组都可以在此注册命令，形成统一的模组管理和操作中心。
## 使用方法
### 打开终端
1. 启动游戏并进入任意场景
2. 将鼠标移至屏幕左下角
3. 会出现一个触发区域，点击即可打开终端
4. 终端从左侧滑入，占据屏幕左半部分
### 选择模组提供者
1. 点击终端底部左侧的"Select Provider"按钮
2. 会显示所有已连接模组的列表
3. 点击任意模组即可选中为当前提供者
4. 也可以在输入框输入 # 加上模组短 ID（如 #816、#dem）快速筛选提供者
### 执行命令
1. 选择提供者后，在底部输入框输入命令
2. 按回车键执行命令
3. 执行结果会显示在上方的可滚动消息区域
4. 使用 help 或 /? 命令查看当前提供者的所有可用命令
