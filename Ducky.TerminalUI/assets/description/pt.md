# Interface de Terminal - Console Universal para Mods
Repositório: https://github.com/ducky7go/Ducky.Terminals
## O que é a Interface de Terminal?
A Interface de Terminal é uma framework de console universal para Escape from Duckkov. Ela fornece uma interface de linha de comandos centralizada onde todos os mods compatíveis podem registrar seus comandos, criando um centro de controle unificado para gestão e operações de mods.
## Como Usar
### Abrir o Terminal
1. Inicie o jogo e entre em qualquer cena
2. Mova seu mouse para o canto inferior esquerdo da tela
3. Uma área de gatilho aparecerá - clique nela para abrir o terminal
4. O terminal desliza da esquerda, ocupando 50% da tela
### Selecionar um Provedor de Mod
1. Clique no botão "Select Provider" no canto inferior esquerdo do terminal
2. Uma lista de todos os mods conectados aparecerá
3. Clique em qualquer mod para selecioná-lo como provedor ativo
4. Alternativamente, digite # seguido do ID curto do mod (ex: #816, #dem) para filtrar provedores rapidamente
### Executar Comandos
1. Após selecionar um provedor, digite comandos no campo de entrada inferior
2. Pressione Enter para executar o comando
3. Os resultados serão exibidos na área de mensagens rolável acima
4. Use help ou /? para ver os comandos disponíveis para o provedor selecionado
