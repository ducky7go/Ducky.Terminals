# Interface Terminal - Interface Console Universelle pour Mods
Dépôt : https://github.com/ducky7go/Ducky.Terminals
## Qu'est-ce que l'Interface Terminal ?
L'Interface Terminal est un framework d'interface console universel pour Escape from Duckkov. Elle fournit une interface de ligne de commande centralisée où tous les mods compatibles peuvent enregistrer leurs commandes, créant ainsi un centre de contrôle unifié pour la gestion et les opérations des mods.
## Comment l'utiliser
### Ouvrir le Terminal
1. Lancez le jeu et entrez dans n'importe quelle scène
2. Déplacez votre souris dans le coin inférieur gauche de l'écran
3. Une zone de déclenchement apparaîtra - cliquez dessus pour ouvrir le terminal
4. Le terminal glisse depuis la gauche, occupant 50% de l'écran
### Sélectionner un Fournisseur de Mod
1. Cliquez sur le bouton "Select Provider" en bas à gauche du terminal
2. Une liste de tous les mods connectés apparaîtra
3. Cliquez sur n'importe quel mod pour le sélectionner comme fournisseur actif
4. Alternativement, tapez # suivi de l'ID court du mod (par exemple, #816, #dem) pour filtrer rapidement les fournisseurs
### Exécuter des Commandes
1. Après avoir sélectionné un fournisseur, tapez des commandes dans le champ de saisie en bas
2. Appuyez sur Entrée pour exécuter la commande
3. Les résultats s'afficheront dans la zone de messages déroulable ci-dessus
4. Utilisez help ou /? pour voir les commandes disponibles pour le fournisseur sélectionné
## Essayez-le
Ce mod fournit uniquement le framework d'interface. Pour expérimenter la fonctionnalité réelle, abonnez-vous à ces mods de démonstration :
- [Demo Terminal Client without Ducky.Sdk](https://steamcommunity.com/sharedfiles/filedetails/?id=3606789962) - Exemple d'intégration manuelle
- [Demo Terminal Client](https://steamcommunity.com/sharedfiles/filedetails/?id=3606789816) - Exemple d'intégration basée sur SDK
Recherchez plus de mods compatibles sur le Steam Workshop ou consultez le dépôt GitHub pour une liste des mods supportés.
## Pour les Développeurs de Mods
Vous voulez intégrer votre mod avec l'Interface Terminal ? Visitez https://github.com/ducky7go/Ducky.Terminals pour des guides d'intégration complets, une documentation API et des exemples de code. Supporte les méthodes d'intégration basées sur SDK (recommandé) et manuelles.
