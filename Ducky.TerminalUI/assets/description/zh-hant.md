# 終端介面 - 模組通用控制台
倉庫地址：https://github.com/ducky7go/Ducky.Terminals
## 什麼是終端介面？
終端介面為《逃離鴨科夫》打造的通用控制台框架。它提供了一個集中的命令列介面，所有相容模組都可以在此註冊指令，形成統一的模組管理和操作中心。
## 使用方法
### 開啟終端
1. 啟動遊戲並進入任意場景
2. 將滑鼠移至畫面左下角
3. 會出現一個觸發區域，點擊即可開啟終端
4. 終端從左側滑入，佔據螢幕左半部分
### 選擇模組提供者
1. 點擊終端底部左側的「Select Provider」按鈕
2. 會顯示所有已連接模組的清單
3. 點擊任意模組即可選中為當前提供者
4. 也可以在輸入框輸入 # 加上模組短 ID（如 #816、#dem）快速篩選提供者
### 執行指令
1. 選擇提供者後，在底部輸入框輸入指令
2. 按回車鍵執行指令
3. 執行結果會顯示在上方的可捲動訊息區域
4. 使用 help 或 /? 指令檢視當前提供者的所有可用指令
